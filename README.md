📱 Responsive Mobile-First Calculator

A sleek, fully interactive, and responsive calculator web application built with HTML, CSS, and vanilla JavaScript. Designed primarily with a mobile-first approach, it looks stunning on phones, tablets, and desktops alike.

✨ Features

- Responsive Modern Design:Beautiful dark-mode user interface inspired by modern smartphone apps.
- Pure Vanilla Stack:** Built entirely with standard HTML5, CSS3 (using CSS Grid), and modern JavaScript without any external frameworks or dependencies.
- Mobile Optimized:** Includes specific mobile-friendly fixes like `aspect-ratio` scaling to keep buttons circular and `-webkit-tap-highlight-color` suppression to remove ugly tap boxes on mobile browsers.
- Full Mathematical Logic:
  - Standard arithmetic operators: Addition (`+`), Subtraction (`-`), Multiplication (`×`), Division (`÷`).
  - Clear Screen functionality (`AC`).
  - Positive/Negative sign toggling (`±`).
  - Percentage calculations (`%`).
  - Floating-point calculations with decimal handling.
  - Safe error handling (e.g., prevents dividing by zero).
